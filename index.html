Credits & Tech Stack

Transparency is key. Here is how this platform was built.

Data Source

All metadata, album artwork, and search results are provided via the public Apple iTunes Search API. By relying on a public, open API, we completely eliminate the need for user accounts or authentication flows.

Architecture

This application is a 100% serverless Single-Page Application (SPA). There is no backend server handling your requests, which means there is no database storing your search history. The requests go directly from your browser to Apple's servers.

UI / UX

Styled completely from scratch utilizing Tailwind CSS for responsive, modern design components. Icons are provided by FontAwesome.
